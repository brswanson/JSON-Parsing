# PizzaAppliction
Simple C# console app for parsing JSON data. It reads a file of pizza toppings and wites out the ranking of each pizza combination by frequency.

Example data:
[
  {
    "toppings": [
      "pepperoni"
    ]
  },
  {
    "toppings": [
      "feta cheese"
    ]
  },
  {
    "toppings": [
      "pepperoni"
    ]
  }
]
